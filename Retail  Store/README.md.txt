# Retail Store Management System

## Overview

This **Retail Store Management System** is a project developed using Object-Oriented Programming (OOP) principles in Python. It simulates core functionalities of a retail store, such as inventory management, sales processing, and customer management. This project is useful for demonstrating OOP principles in a real-world application setting, including encapsulation, inheritance, and polymorphism.

## Project Structure

The project is organized as follows:

Retail-Store/ ├── README.md # Project overview and instructions ├── store.py # Main script containing the RetailStore class ├── inventory.py # Inventory class to manage products and stock levels ├── customer.py # Customer class for handling customer details ├── sales.py # Sales class to process transactions and manage sales ├── data/ # Folder with initial data files for products and customers │ ├── products.csv # Product details (ID, name, price, stock) │ └── customers.csv # Customer information (ID, name, email) ├── reports/ # Folder to save sales reports and inventory summaries └── requirements.txt # List of dependencies

## Features

- **Inventory Management**: Track products, prices, and stock levels.
- **Customer Management**: Maintain a list of customers with basic information.
- **Sales Processing**: Process sales, update stock, and generate receipts.
- **Reporting**: Generate reports on inventory and sales performance.
